# Análise de Dados de Varejo com KDD

##  Descrição
Projeto acadêmico focado na análise de dados de vendas de varejo,
utilizando a metodologia KDD para extração de conhecimento.

##  Objetivo
Identificar padrões de consumo e segmentar clientes para apoiar
decisões estratégicas de negócio.

##  Metodologia
- KDD (Knowledge Discovery in Databases)
- Limpeza e preparação de dados
- Análise exploratória
- Clustering
- Regressão

##  Tecnologias
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Jupyter Notebook

##  Resultados
- Identificação de clusters de clientes
- Relação entre variáveis de venda
- Insights para tomada de decisão

Explicação das pastas e arquivos.
